# Posts-microservice

A microservice component for the SnapMsg API.

## Getting Started

### Dependencies

Before you begin, ensure you have met the following requirements, which are also listed in the `requirements.txt` file:

- **FastAPI:** `0.103.1`
- **Uvicorn:** `0.23.2`
- **DataDog:** `0.47.0`
- **Pydantic:** `2.3.0`
- **MongoDb:** `2.0.20`
- **Psycopg2:** `2.9.7`
- **Pytest:** `7.2.1`
- **HTTPX:** `0.23.3`
- **Docker:** `1.0`


### Executing program
```sh
make help
```
